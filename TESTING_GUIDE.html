Go to Firebase Console → Authentication → Users and create 4 test accounts with Email/Password.
Go to Firestore → userRoles collection. Create 4 documents (one per user UID) with field role:

receptionist@test.com → role: "receptionist"

doctor@test.com → role: "doctor"

8. WhatsApp Integration Testing

Setup Required
Configure Twilio credentials in .env.local (TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN, TWILIO_WHATSAPP_FROM).
For local testing: run ngrok http 3000 and set webhook URL in Twilio sandbox settings to https://YOUR_NGROK_URL/api/whatsapp/webhook.
Tests
Issue a token with a real phone number → Call Next.
WhatsApp message received: "Your token #XXX is being called".
Create in-patient card with phone → click WhatsApp button.
Card details received via WhatsApp.
Text "HELP" to the Twilio sandbox number.
Bot replies with menu of commands.
Text "APPOINTMENT" to the number.
Bot replies with appointment info or hospital contact.
Text "STATUS" to the number.
Bot replies with queue check link.
Generate an invoice → click "Request Review via WhatsApp".
Review link received via WhatsApp.

9. Final Go-Live Checklist

Before handing over to client
  • Hospital name, phone, address updated in .env.local
  • Firebase project created for the client (separate from test)
  • Firestore rules published (simple if request.auth != null rules)
  • Storage rules published
  • Google Sign-In enabled in Firebase Auth
  • Admin account created and userRoles document set in Firestore
  • All 4 staff roles created and tested
  • Seed data cleared after testing (/admin/settings → Delete All Data)
  • Logo uploaded to /public/images/logo.jpg
  • Doctor profiles updated with real names and credentials
  • Hospital address updated in src/data/locations.ts
  • Emergency phone number confirmed correct
  • Twilio account created (optional for WhatsApp)
  • Domain registered and DNS configured
  • Deployed to Vercel (connect GitHub repo → auto-deploy)
  • All env vars added to Vercel project settings
  • Staff trained on admin panel, desk, and pharmacy
