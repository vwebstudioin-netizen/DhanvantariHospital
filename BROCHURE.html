Smart Patient Token Queue

5-stage flow with WhatsApp notification at each step

WAITING

Token Issued

Reception issues token. Patient gets slip with number.

CALLED

Call Next

Token called. WhatsApp sent to patient automatically.

SERVING

Start Consultation

Doctor/staff clicks Start — consultation begins.

DONE

Done → Next

Mark complete. Next patient auto-called.

SKIP

Skip / No Show

Patient absent — skip or mark no-show. Next called.

TV Display: Full-screen waiting room monitor shows current token in real-time with audio chime. Patients can also check their queue position from their phone at /queue.

Auto-message sent when patient's token is called. Includes token number and hospital name.
